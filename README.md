## Features

## Tech Stack

**Language:** Python  
**Bot Framework:** aiogram / pyTelegramBotAPI (use whichever you prefer)  
**Weather API:** OpenWeatherMap (or any compatible API)  
**Dependencies:** requests, python-dotenv  
**Environment:** Works on Linux/Windows/macOS



## Installation

1. Clone the repository:

```bash
git clone https://github.com/Crack10110/weather-telegram-bot
cd weather-telegram-bot
```
2. Install dependencies:

```bash
pip install -r requirements.txt
```

3. Create a `.env` file based on `.env.example` and add your tokens.

## Environment Variables

To run this project, you need the following environment variables:

`BOT_TOKEN` ‚Äî your Telegram Bot API token  
`WEATHER_API_KEY` ‚Äî your API key from OpenWeather (or any other weather provider)
